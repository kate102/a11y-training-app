Using VoiceOver

To Set Up VoiceOver

System Preferences > Accessibility > VoiceOver > Enable VoiceOver

To Turn VoiceOver On or Off

command key + F5

To Stop VoiceOver

control key

To Get Into A Site

control key + option key + shift key + down arrow

To Move Around A Page

control key + option key + (right arrow or left arrow)

To Select Something

control key + option key + space bar

To Navigate By Headings

control key + option key + command key + 'H'

To Open The Rotor

control key + option key + 'U'

To Speed Voice Up or Down

control key + option key + command key + (up arrow or down arrow)
